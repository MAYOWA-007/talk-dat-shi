# Image-Generated Minimal Logo Direction

This folder contains a cleaner generated logo direction with the cheap shine removed.

- `flow-minimal-logo-transparent.png`: 2048 x 2048 transparent PNG candidate
- `flow-minimal-logo-preview-light.jpg`: preview on light background
- `flow-minimal-logo-preview-dark.jpg`: preview on dark background
- `flow-minimal-logo-preview-checker.jpg`: preview on checkerboard
- `flow-minimal-logo-size-preview.jpg`: 256 / 128 / 64 / 32 / 16 px readability check

This is not wired into the app icon set yet. Once approved, replace `knight_flow/assets/app_icon.*`, `logo.*`, and `favicon.*`, then rebuild the EXE and installer.
