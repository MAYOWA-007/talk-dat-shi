# Talk Stone

Talk Stone is the official Talk Dat Shi app mark. It uses a black roman-clay / limewash surface with the Flow waveform material preserved inside the bars.

- `talk-stone-transparent.png`: 2048 x 2048 transparent PNG source
- `talk-stone-preview-light.jpg`: preview on light background
- `talk-stone-preview-dark.jpg`: preview on dark background
- `talk-stone-preview-checker.jpg`: preview on checkerboard
- `talk-stone-size-preview.jpg`: 256 / 128 / 64 / 32 / 16 px readability check

The colored waveform bars intentionally preserve the Flow material while the black tile uses a restrained mineral texture. The generated app icon, favicon, installer icon, desktop shortcut icon, tray icon, and logo files are derived from this source.
